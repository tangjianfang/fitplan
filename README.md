# 资深教练 · 智能健身饮食方案

纯静态网页工具：输入一段自然语言（如「男 32 岁 178cm 78kg 体脂20% 想增肌 一周练4次」），自动按《运动饮食指南》文档严格计算并生成可打印为 A4 PDF / 保存为图片的高标准方案。

## 核心特性
- **极简输入**：一段话即可，规则+正则抽取（性别 / 年龄 / 身高 / 体重 / 体脂 / 目标 / 训练频率 / 强度 / 餐次），常用表达 ≥90% 命中
- **严格按文档**：BMR 公式、活动系数、目标热量计算（含安全下限）、宏量比、餐次比例（含练后餐）、膳食纤维、水合、烹饪与用油、月度复盘 100% 来自文档
- **真实食材**：内置文档附录 7 张表共约 100 种食物（每 100 g 营养素），按 减脂/增肌 选品规则自动匹配并反算克数
- **导出方便**：A4 打印 → 保存为 PDF；或一键导出 PNG 图片，方便手机查看

## 本地预览
直接用浏览器打开 `index.html` 即可。

## 部署到 GitHub Pages
1. 推送本仓库到 GitHub
2. 仓库设置 → Pages → Build and deployment → Source 选择 **GitHub Actions**
3. 推送到 `main`/`master` 分支即自动部署

## 文件结构
```
index.html
assets/
  css/styles.css
  js/data.js     # 公式 / 系数 / 食物数据库（来自文档）
  js/extract.js  # 自然语言抽取
  js/calc.js     # 计算引擎
  js/render.js   # 报告渲染
  js/app.js      # 入口
.github/workflows/deploy.yml
```

## 数据来源
所有计算公式、活动系数、热量目标算法、宏量比、餐次比例、膳食纤维、水合规则、烹饪与用油规则、食物营养数据均直接来自仓库内 `运动饮食指南.docx` 文档。
